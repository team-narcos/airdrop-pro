================================================================================
TROUBLESHOOTING
================================================================================

Problem: "adb is not recognized"
Solution: You need to install Android Platform Tools
   Download from: https://developer.android.com/tools/releases/platform-tools

Problem: "device not found"
Solution: 
   - Connect phone via USB
   - Enable USB debugging in phone settings
   - Allow USB debugging when popup appears on phone

Problem: Can't find APK
Solution:
   - Check both locations listed in STEP 1
   - Or search for "app-release.apk" in Windows search

Problem: Logs not showing
Solution:
   - Make sure phone is connected
   - Run: adb devices (should show your device)
   - Try the command again
